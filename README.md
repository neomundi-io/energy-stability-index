# energy-stability-index
Public methodological notes on NeoMundi Energy Stability Index for AI runtime governance.

ESI est un indice composite de gouvernance runtime.

Il vise à agréger plusieurs dimensions observables :
- S : stabilité de gouvernance ;
- P : densité informationnelle ;
- Q : énergie informationnelle ;
- C : cohérence sémantique.

ΔG peut être utilisé comme signal de variation ou de transition permettant d’interpréter l’évolution de la stabilité au cours du temps.

Le score peut être présenté sur une échelle de 0 à 100 pour faciliter la lecture opérationnelle.

Les coefficients, seuils, pondérations, règles d’intégration et paramètres de production ne sont pas documentés dans ce dépôt public.
